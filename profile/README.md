## 📖 About Image DPI Converter

**Image DPI Converter** is a browser-based local image DPI checker and converter tool built for designers, print shops, ecommerce sellers, photographers, and document workflows.

The tool runs fully inside your browser, so your images stay private on your own device with no server upload required.

It helps users quickly:

- check current image DPI metadata
- change image DPI to 72, 150, 300, 600, or custom
- prepare images for professional printing
- resize images in cm, mm, inch, and pixels
- export JPG, PNG, or WebP
- batch process multiple images
- keep original quality with metadata-only DPI updates

One uploaded image can instantly become print-ready for multiple use cases.
